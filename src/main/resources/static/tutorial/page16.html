Окружность

Окружностью называется геометрическое место точек плоскости, равноудалённых от данной точки, называемой центром окружности.

Основные свойства окружности

  1. 1. Диаметр, перпендикулярный хорде, делит хорду и стягиваемые ею дуги пополам.
  2. 2. Диаметр, проходящий через середину хорды, не являющейся диаметром, перпендикулярен этой хорде.
  3. 3. Серединный перпендикуляр к хорде проходит через центр окружности.
  4. 4. Равные хорды удалены от центра окружности на равные расстояния.
  5. 5. Хорды окружности, удалённые от центра на равные расстояния, равны.
  6. 6. Окружность симметрична относительно любого своего диаметра.
  7. 7. Дуги окружности, заключённые между параллельными хордами, равны.
  8. 8. Из двух хорд больше та, которая менее удалена от центра.
  9. 9. Диаметр есть наибольшая хорда окружности.

Замечательные свойства окружности

  1. 1. Геометрическое место точек M, из которых отрезок AB виден под прямым углом (∠AMB = 90°), есть окружность с диаметром AB без точек A и B
  2. 2. Геометрическое место точек M, из которых отрезок AB виден под острым углом (∠AMB < 90°), есть внешность круга с диаметром AB без точек прямой AB.
  3. 3. Геометрическое место точек M, из которых отрезок AB виден под тупым углом (∠AMB > 90°), есть внутренность круга с диаметром AB без точек отрезка AB
  4. 4. Геометрическое место точек, из которых данный отрезок виден под данным углом, есть две дуги равных окружностей (без концов этих дуг).

Задачи с решениями

Колесо имеет 32 спицы. Найдите величину угла (в градусах), который образуют две соседние спицы.

Решение

32 спицы делят круг на 32 части.

$360/32=11.25$

Ответ: 11,25.
